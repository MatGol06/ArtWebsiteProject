# 🎨 The Art Gallery of You

An interactive storytelling website built with React that transforms a simple confession into a digital art gallery experience.

- [Design System](./DESIGN_SYSTEM.md)
- [Roadmap](./ROADMAP.md)
- [Component Structure](./COMPONENT_STRUCTURE.md)
- [Animation Guide](./ANIMATION_GUIDE.md)

## 🛠️ Tech Stack
- **Frontend:** React + Vite
- **Styling:** Tailwind CSS v4
- **Animation:** GSAP (ScrollTrigger) & Framer Motion
- **Scrolling:** Lenis Smooth Scroll
- **Icons:** Lucide React

## 🚀 Getting Started
```bash
npm install
npm run dev
```
